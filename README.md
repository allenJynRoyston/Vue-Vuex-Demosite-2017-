# Better Energy Mockup

### Install instructions:
```sh
$ git clone https://github.com/allenRoyston/mockVueDemo.git
$ cd mockVueDemo
$ npm install npm
```

### Development build
```sh
$ npm run dev
```

### Bundle instruction
```sh
// this will bundle everything with webpack
$ gulp build
// this will present the final version of the site
$ gulp
```
